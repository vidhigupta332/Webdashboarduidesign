# Web dashboard ui design
Digital Guard is a frontend dashboard UI for an Edge-AI agricultural pest monitoring system. It visualizes crop health, pest alerts, geospatial heatmaps, organic remedy suggestions, and device status to help farmers detect threats early and protect crops sustainably.
Dashboard Features:
Overview dashboard with system status cards
Real-time pest alert visualization
Crop health indicators
Geospatial heatmap view for pest activity
Organic pest remedy recommendation UI
Edge device status and battery monitoring UI
